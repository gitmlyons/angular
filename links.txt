1. node js must be downloaded and installed first
2. npm install -g typescript
3. npm install -g @angular/cli
4. ng new my-app
5. ng serve
6. install VSCode and simply open that directory
7. Debug using console/sources in Google Chrome under the webpack folder
8. npm install other things like bootstrap and font awesome, favourite controls 
9. Adding featurs - either just add in creating code files directly or use cli see - https://www.sitepoint.com/ultimate-angular-cli-reference/
10. Rebuilds are automatic under dev changes / webpack while ng serve is running
11. Testing - to do but a bit messy with Karma having own version of compiling IMHO
12. Prod build is a matter of setting that mode and packing for release
13. Code example in my-app in this repository
